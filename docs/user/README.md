## Quick Start

1. **Sign in** with your email
2. **Create an itinerary** by clicking "New Itinerary"
3. **Chat with the AI** to plan your trip
4. **Refine your plans** through conversation
5. **View your itinerary** in List, Calendar, or Map view

## Need Help?

- Click the **Help** button (?) in the header for documentation
- Ask the AI assistant directly in any chat
- Type "help" or "how do I..." to get guidance
