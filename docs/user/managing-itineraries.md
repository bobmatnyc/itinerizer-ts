# Managing Itineraries

Learn how to organize, edit, and manage your travel plans effectively.

## View Modes

### Itinerary View (Default)
The main view shows your itinerary as a list of segments (activities, accommodations, transportation, etc.).

- **Segments**: Each item on your itinerary
- **Timeline**: Activities ordered by date and time
- **Details**: Click any segment to see full information

### Calendar View
See your itinerary on a calendar grid.

- **Daily overview**: All activities for each day
- **Visual timeline**: Easy to spot gaps or overlaps
- **Quick navigation**: Jump to specific dates

### Map View
Visualize your trip geographically.

- **Location markers**: See where each activity takes place
- **Route visualization**: Understand distances between locations
- **Area context**: Discover nearby attractions

### Travelers View
Manage who's joining your trip.

- **Add travelers**: Include family, friends, or group members
- **Traveler details**: Names, contact info, special needs
- **Per-person planning**: Assign activities to specific travelers

## Editing Itineraries

### Edit Metadata
Click the itinerary title to edit:
- Title and description
- Start and end dates
- Trip type (leisure, business, mixed)
- Tags for organization

### Edit Segments
Click any segment to:
- Change times and dates
- Update location details
- Add notes and reminders
- Set segment type (activity, accommodation, transport, meal)

### Using AI to Edit
Instead of manual editing, ask the AI:
- "Change the dinner reservation to 7pm"
- "Move tomorrow's activities to next Thursday"
- "Add a note about bringing sunscreen"

## Organizing Your Collection

### Search and Filter
- Use the search bar to find specific itineraries
- Filter by date, destination, or status

### Itinerary Status
- **Draft**: Work in progress
- **Planned**: Ready for the trip
- **Active**: Currently traveling
- **Completed**: Trip finished

### Deleting Itineraries
- Click the delete button on any itinerary
- Confirm the deletion (cannot be undone)
- Consider exporting first if you want a backup
