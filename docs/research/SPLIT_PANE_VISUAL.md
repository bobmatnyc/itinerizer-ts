## Key Implementation Details

### SvelteKit Layout System
- `+layout.svelte` wraps all child routes
- Provides persistent left pane
- Uses `{@render children?.()}` for Svelte 5

### URL-Based State
- No local "selected" state variable
- URL is source of truth
- $page.params.id drives selection
- Browser back/forward works automatically

### Conditional Rendering
```svelte
{#if showChatSidebar}
  <ChatPanel ... />
{/if}

{#if isPaneVisible}
  <VisualizationPane ... />
{/if}
```

### Responsive Design
- CSS media query at 768px
- Desktop: `flex-direction: row`
- Mobile: `flex-direction: column`
- Maintains all functionality on mobile
