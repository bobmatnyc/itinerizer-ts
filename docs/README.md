# Documentation

This directory contains all project documentation organized by purpose.

## Directory Structure

### `/research/` - Implementation Notes & Analysis
Implementation details, feature documentation, bug analyses, and technical investigations. These documents capture the evolution of features and technical decisions made during development.

### `/developer/` - Deployment & Development Guides
Deployment procedures, environment setup, and operational guides for developers and operators.

### `/architecture/` - System Architecture
High-level architectural documentation, system design documents, and architectural decision records (ADRs).

### `/user/` - User Guides
End-user documentation, tutorials, and how-to guides for using the Itinerizer application.
