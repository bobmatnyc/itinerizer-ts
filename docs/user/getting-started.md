# Getting Started with Itinerizer

Welcome to Itinerizer, your AI-powered travel planning assistant!

## Creating Your First Itinerary

1. **Click "New Itinerary"** in the sidebar to start a new travel plan
2. **Enter basic details**: Give your trip a title and set the date range
3. **Start chatting**: Use the AI Trip Designer to describe your ideal trip

## Importing Existing Plans

If you already have travel plans, you can import them:

### Import PDF
- Click "Import PDF" in the sidebar
- Select a PDF containing your itinerary (hotel confirmations, tour bookings, etc.)
- The AI will parse and organize the information automatically

### Import Text
- Click "Import Text" for copied text from emails or other sources
- Paste your itinerary text and let the AI structure it

## Understanding the Interface

### Sidebar (Left Panel)
- **Itinerary List**: All your travel plans
- **New Itinerary**: Create a new trip
- **Import Options**: PDF and text import

### Main Panel (Right Side)
- **Itinerary View**: Details of the selected trip
- **Chat Interface**: Talk to the AI Trip Designer
- **View Modes**: Switch between List, Calendar, and Map views

### Header (Top)
- **Profile**: Configure your settings and API keys
- **Help**: Access this help documentation
- **Logout**: Sign out of your account
